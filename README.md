# Bedrock Weather & Hazard App

A Flutter application for real‑time weather, earthquake, and crowd‑sourced hazard alerts in Abbottabad, Pakistan. Features:
- Firebase/Firestore sync with robust offline fallbacks.
- Admin control panel for moderation, user management and system broadcasts.
- System advisory banner, dynamic map UI, and ML‑powered weather weighting.
- Comprehensive comments added throughout the codebase.

## Getting Started
```bash
flutter pub get
flutter run
```

## Repository Structure
- `lib/` – core, data, domain, presentation layers.
- `test/` – widget tests.
- `todo.md` – implementation plan and future tasks.
- `README.md` – this file.
