### 1. Project Overview

* **App Name:** Bedrock Abbottabad (Global Weather & Hazard Network)
* **Purpose:** Bedrock is an offline-first, crowd-sourced weather intelligence and localized hazard alert application. It allows users in mountain sectors and low-connectivity regions (specifically modeled for Abbottabad, Pakistan) to report and view real-time hazards like landslides, flooding, road accidents, and earthquakes. The app uses a 3-tier offline fallback architecture (Live Firestore $\rightarrow$ Local Cache $\rightarrow$ Static Assets) and features custom vector graphics painters to render map paths, sunrise trajectories, and weather spline graphs completely offline without requiring massive raster data downloads.
* **Platform:** Flutter (Android & iOS)

##### 3. Subcollection Name: `hazards/{id}/votes/{uid}`
Tracks individual votes on a hazard to ensure vote deduplication.

| Field Name | Data Type | Purpose / Description |
| :--- | :--- | :--- |
| `uid` (Document ID) | `String` | UID of the voting user (Foreign Key linking to `users`). |
| `vote` | `String` | Vote state cast by user (`up` or `down`). |
| `votedAt` | `Timestamp` | Timestamp when the vote was submitted. |

#### D. Database Relationships
* **User-to-Hazard (One-to-Many):** A user doc in `users/{uid}` can have many reports in the `hazards` collection. The linkage is established by setting the `reporterId` in the hazard document equal to the user's `uid`.
* **Hazard-to-Votes (One-to-Many Subcollection):** Every hazard contains a subcollection of votes. The document ID in the subcollection is the voter's `uid`, preventing a user from voting multiple times on the same hazard report.
* **Social Connections (Many-to-Many):** User relations are handled using lists of UIDs in the `followers` and `following` array fields within each user document.

// Location: lib/data/dto/hazard_dto.dart
class HazardDto {
  final String id;
  final HazardType type;
  final String description;
  final int upvotes;
  final int downvotes;
  final double trustScore;
  final String reporterName;
  final ReputationTier reporterTier;
  final DateTime reportedAt;
  final double latitude;
  final double longitude;
  final String geohash;
  final String reporterId;
  final String? imageUrl;

  HazardDto({
    required this.id,
    required this.type,
    required this.description,
    required this.upvotes,
    required this.downvotes,
    required this.trustScore,
    required this.reporterName,
    required this.reporterTier,
    required this.reportedAt,
    required this.latitude,
    required this.longitude,
    required this.geohash,
    required this.reporterId,
    this.imageUrl,
  });

  factory HazardDto.fromFirestore(Map<String, dynamic> data, String documentId) {
    return HazardDto(
      id: documentId,
      type: _parseHazardType(data['type'] as String?),
      description: data['description'] as String? ?? '',
      upvotes: data['upvotes'] as int? ?? 0,
      downvotes: data['downvotes'] as int? ?? 0,
      trustScore: (data['trustScore'] as num?)?.toDouble() ?? 0.0,
      reporterName: data['reporterName'] as String? ?? 'Anonymous',
      reporterTier: _parseReputationTier(data['reporterTier'] as String?),
      reportedAt: (data['reportedAt'] as Timestamp?)?.toDate() ?? DateTime.now(),
      latitude: (data['latitude'] as num?)?.toDouble() ?? 0.0,
      longitude: (data['longitude'] as num?)?.toDouble() ?? 0.0,
      geohash: data['geohash'] as String? ?? '',
      reporterId: data['reporterId'] as String? ?? '',
      imageUrl: data['imageUrl'] as String?,
    );
  }

  Map<String, dynamic> toFirestore() {
    return {
      'type': type.name,
      'description': description,
      'upvotes': upvotes,
      'downvotes': downvotes,
      'trustScore': trustScore,
      'reporterName': reporterName,
      'reporterTier': reporterTier.name,
      'reportedAt': Timestamp.fromDate(reportedAt),
      'latitude': latitude,
      'longitude': longitude,
      'geohash': geohash,
      'reporterId': reporterId,
      'imageUrl': imageUrl,
    };
  }
}

#### B. CRUD Operations (`firestore_hazard_datasource.dart`)
This snippet demonstrates the atomic Firestore transaction and batch writes used to submit reports and submit votes securely without concurrency errors.

```dart
// Location: lib/data/datasources/remote/firestore_hazard_datasource.dart
class FirestoreHazardDataSource {
  // Collection Reference
  CollectionReference<Map<String, dynamic>> get _hazardsCollection =>
      FirebaseFirestore.instance.collection('hazards');

  /// Submits a new hazard report. Increments user's totalReports atomically.
  Future<String> submitHazard(HazardDto hazard, String userId) async {
    final firestore = FirebaseFirestore.instance;
    final batch = firestore.batch();

    final docRef = _hazardsCollection.doc();
    final data = hazard.toFirestore();
    data['reporterId'] = userId;
    data['status'] = 'active';
    data['expiresAt'] = Timestamp.fromDate(
      hazard.reportedAt.add(const Duration(hours: 6)),
    );

    // Write hazard report
    batch.set(docRef, data);

    // Update user statistics atomically in the same batch
    final userRef = firestore.collection('users').doc(userId);
    batch.update(userRef, {'totalReports': FieldValue.increment(1)});

    await batch.commit();
    return docRef.id;
  }
}
```

#### D. Feature Implementation (`accelerometer_service.dart`)
This class implements the accelerometer force threshold stream subscription to prompt a emergency warning during sudden decelerations.

```dart
// Location: lib/core/services/accelerometer_service.dart
import 'dart:async';
import 'dart:math' as math;
import 'package:sensors_plus/sensors_plus.dart';

class AccelerometerService {
  StreamSubscription<UserAccelerometerEvent>? _subscription;
  DateTime? _lastPromptTime;
  final double shockThreshold = 25.0; // ~2.5G shock threshold

  void startListening({required Function() onShockDetected}) {
    try {
      _subscription?.cancel();
      _subscription = userAccelerometerEvents.listen(
        (UserAccelerometerEvent event) {
          final double magnitude = math.sqrt(
            event.x * event.x + event.y * event.y + event.z * event.z,
          );

          if (magnitude > shockThreshold) {
            final now = DateTime.now();
            if (_lastPromptTime == null ||
                now.difference(_lastPromptTime!).inSeconds > 10) {
              _lastPromptTime = now;
              onShockDetected();
            }
          }
        },
        onError: (err) {},
        cancelOnError: false,
      );
    } catch (_) {}
  }

  void stopListening() {
    _subscription?.cancel();
    _subscription = null;
  }
}
```
